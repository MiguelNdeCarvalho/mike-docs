[**Speedtest exporter**][1] is a [**Prometheus**][2] exporter written in **Python** using the official [Speedtest CLI][1] made by **Ookla**.

It will measure your:

- Download Speed
- Upload Speed
- Ping
- Jitter

[2]: https://prometheus.io/
